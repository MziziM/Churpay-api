# Brand Assets

Replace these files to rebrand the admin portal:

- `churpay-logo.svg` (full logo, preferred)
- `churpay-logo.png` (legacy fallback)
- `churpay-mark.svg` (compact icon)

The admin portal and favicon files reference these assets from:

- `/assets/brand/churpay-logo.svg`
- `/assets/brand/churpay-mark.svg`

If you replace `churpay-logo.svg` / `churpay-logo.png`, also regenerate these files in `public/`:

- `favicon.ico`
- `favicon.png`
- `apple-touch-icon.png`
